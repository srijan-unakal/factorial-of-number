\# Factorial Calculator (Java)



\## Overview



This project contains a simple Java program that calculates the factorial of a number entered by the user.

The program runs in the terminal and allows the user to perform multiple calculations until they choose to exit.



It also handles invalid input such as negative numbers and informs the user appropriately.



\## Features



\* Accepts user input from the terminal

\* Calculates the factorial using a `for` loop

\* Detects and handles negative numbers

\* Allows repeated calculations using a loop

\* Simple and beginner-friendly Java implementation



\## How the Program Works



1\. The program asks the user to enter a number.

2\. If the number is negative, the program displays an error message.

3\. If the number is valid, the factorial is calculated using a loop.

4\. After displaying the result, the user is asked whether they want to continue (`Y/N`).

5\. The program repeats until the user chooses to exit.



\## Example Output



```

This program prints factorial of given number

Enter a number : 5

The factorial of the number is : 120

Do you want to continue : Y/N

```



\## How to Run



Compile the program:



```

javac factorial.java

```



Run the program:



```

java factorial

```



\## Concepts Used



\* Java loops (`for`, `while`)

\* Conditional statements (`if`)

\* User input using `Scanner`

\* Loop control (`break`, `continue`)
